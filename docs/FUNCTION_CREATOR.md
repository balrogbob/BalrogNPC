# Function Creator

**Build custom rAthena functions**

---

## Overview

The Function Creator helps you build custom functions with:
- Parameter definitions
- Return type selection
- Code templates
- Live preview

---

## Quick Start

1. Click **rAthena Tools → New Function**
2. Enter function name
3. Add parameters (optional)
4. Choose return type
5. Use template or write custom code
6. Click **Insert Function**
