# Syntax Highlighting

**Colorize rAthena scripts and YAML databases**

---

## Overview

BalrogNPC includes syntax highlighting for rAthena script files and YAML database files, making code easier to read and understand.

---

## Enabling Syntax Highlighting

**Three modes available:**

### Script Mode
- **Menu:** Syntax → Script
- **For:** `.npc` files, rAthena scripts
- **Colors:** Commands, strings, comments, numbers

### Database Mode
- **Menu:** Syntax → Database  
- **For:** `.yml`, `.yaml` files
- **Colors:** Keys, values, strings, comments

### None (Disabled)
- **Menu:** Syntax → None
- **For:** Plain text editing
- **Colors:** All text same color

---

## Color Customization

**Edit syntax colors:**

1. Click **Syntax → Edit Colors...**
2. Select element to customize
3. Click **Foreground** or **Background** button
4. Choose color from picker
5. Click **Save Color**
6. Changes apply immediately

**Elements you can customize:**
- Keywords (commands)
- Strings (text in quotes)
- Comments (// and /* */)
- Numbers
- Background color
