# BalrogNPC Keyboard Shortcuts

**Complete reference of all keyboard shortcuts**

---

## File Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+N` | New | Create a new empty file |
| `Ctrl+O` | Open | Open an existing file |
| `Ctrl+S` | Save | Save current file |
| *(none)* | Save As | Save with new name (File menu) |

---

## Edit Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+Z` | Undo | Undo last action |
| `Ctrl+X` | Cut | Cut selection to clipboard |
| `Ctrl+C` | Copy | Copy selection to clipboard |
| `Ctrl+V` | Paste | Paste from clipboard |
| `Delete` | Delete | Delete selection (Edit menu) |
| `Ctrl+A` | Select All | Select entire document |

---

## Search & Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+F` | Find & Replace | Open Find & Replace dialog |
| `Ctrl+G` | Go To Line | Jump to specific line number |
| `Return` (in Find) | Find Next | Find next occurrence |
| `Shift+Return` (in Find) | Find Previous | Find previous occurrence |

---

## Help & Documentation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `F1` | View Documentation | Open built-in documentation viewer |
