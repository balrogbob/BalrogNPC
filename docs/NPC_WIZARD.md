# NPC Wizard

**Step-by-step NPC creation**

---

## Overview

The NPC Wizard guides you through creating NPCs with a 5-step process:

1. **Name and Location** - Set NPC name, map, coordinates, direction
2. **Appearance** - Choose sprite with preview
3. **NPC Type** - Select type (Dialog, Quest, Shop, etc.)
4. **Dialog Actions** - Build dialog sequence visually
5. **Confirmation** - Review and insert

---

## Quick Start

1. Click **rAthena Tools → NPC Wizard...**
2. Follow each step
3. Click **Next** to proceed
4. Click **Insert** when done
