# YAML Validator

**Validate rAthena YAML database files**

---

## Overview

The YAML Validator checks database files for:
- Syntax errors
- Structure issues
- Field type problems
- Schema compliance

---

## Requirements

**Install PyYAML:**
```bash
pip install pyyaml
```

---

## Quick Start

1. Open a `.yml` or `.yaml` file
2. Click **rAthena Tools ? Validate YAML Database**
3. Review results in three tabs:
   - Errors (must fix)
   - Warnings (should fix)
   - Suggestions (optional)

---

## Supported Files

- `item_db.yml` - Item database
- `mob_db.yml` - Monster database
- `quest_db.yml` - Quest database
- Other rAthena YAML databases
