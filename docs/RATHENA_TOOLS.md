# rAthena Tools Overview

**Complete toolkit for rAthena script development**

---

## Overview

BalrogNPC includes powerful integrated tools for creating and validating rAthena scripts. Access all tools through the **rAthena Tools** menu.

---

## Available Tools

### 1. New NPC Script (NPC Wizard)
- **Menu:** rAthena Tools → New NPC Script
- **Purpose:** Create NPCs step-by-step with guided wizard
- **Features:** Name, location, sprite, type, dialog actions
- See: [User Guide](USER_GUIDE.md#1-new-npc-script)

### 2. New Function
- **Menu:** rAthena Tools → New Function
- **Purpose:** Create custom functions with templates
- **Features:** Parameters, return types, code templates
- See: [User Guide](USER_GUIDE.md#2-new-function)

### 3. NPC Wizard
- **Menu:** rAthena Tools → NPC Wizard...
- **Purpose:** Full-featured NPC creation wizard
- **Features:** 5-step guided process
- See: [User Guide](USER_GUIDE.md#3-npc-wizard)

### 4. Dialog Builder
- **Menu:** rAthena Tools → Dialog Builder
- **Purpose:** Visual dialog sequence builder
- **Features:** Messages, menus, branches, actions
- See: [User Guide](USER_GUIDE.md#4-dialog-builder)

### 5. Validate Script
- **Menu:** rAthena Tools → Validate Script
- **Purpose:** Check scripts for errors
- **Features:** Errors, warnings, auto-fix
- See: [User Guide](USER_GUIDE.md#5-validate-script)

### 6. Validate YAML Database
- **Menu:** rAthena Tools → Validate YAML Database
- **Purpose:** Validate YAML database files
- **Features:** Syntax check, structure verification
- **Requires:** PyYAML (`pip install pyyaml`)
- See: [User Guide](USER_GUIDE.md#6-validate-yaml-database)

### 7. Insert Quick NPC
- **Menu:** rAthena Tools → Insert Quick NPC
- **Purpose:** Insert pre-built NPC templates
- **Features:** 10 templates, customizable
- See: [User Guide](USER_GUIDE.md#7-insert-quick-npc)

---

## Quick Start

**Create your first NPC in 2 minutes:**

1. Click **rAthena Tools → Insert Quick NPC**
2. Select "Simple Dialog"
3. Enter NPC name
4. Set location and sprite
5. Click "Insert NPC"
6. Save your file!
