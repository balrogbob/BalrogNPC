# Quick NPC Templates

Pre-built templates to speed up NPC creation.

---

## Templates
- Basic shop NPC
- Warp NPC
- Quest giver
- Dialogue-only NPC

## How to Use
1. Open an existing script or new file.
2. Insert a template from the menu.
3. Adjust names, locations, and items.
4. Validate with `Script Validator`.

## Tips
- Keep labels unique.
- Use comments to document behavior.
