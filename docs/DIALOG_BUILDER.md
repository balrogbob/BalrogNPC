# Dialog Builder

**Visual dialog sequence builder**

---

## Overview

The Dialog Builder lets you create complex NPC conversations visually:

**Available Actions:**
- Messages - Display text
- Next/Close - Dialog buttons
- Menus - Player choices with branches
- Script commands - Custom logic
- Warps - Teleport player
- Items - Check, give, remove

---

## Quick Start

1. Click **rAthena Tools → Dialog Builder**
2. Click action buttons to add steps
3. Reorder with ↑↓ buttons
4. Preview script on right
5. Click **Insert into Script**
