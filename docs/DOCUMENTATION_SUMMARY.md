  - Any text editor

  - VS Code with preview
  - Markdown viewers/browsers
  - Can be converted to PDF/HTML

- Code blocks with syntax highlighting
- Tables for structured data
- Lists for sequential information

## Documentation Maintenance

### Keeping Docs Updated

When adding features:
1. Update relevant guide (USER_GUIDE.md)
2. Add examples (SCRIPT_EXAMPLES.md if applicable)
3. Update shortcuts (KEYBOARD_SHORTCUTS.md if new)
4. Update FAQ (FAQ.md if common question)
5. Update INDEX.md navigation

### Version Tracking
- Documentation version matches BalrogNPC version
- Major changes documented in each file
- Last updated date in headers (where applicable)
