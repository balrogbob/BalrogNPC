# Script Validator

**Validate and auto-fix rAthena scripts**

---

## Overview

The Script Validator checks your scripts for:
- **Errors** - Must fix (missing semicolons, unbalanced brackets)
- **Warnings** - Should fix (indentation, style)
- **Best Practices** - Suggestions (optional improvements)

---

## Quick Start

1. Open or write a script
2. Click **rAthena Tools → Validate Script**
3. Review issues in tabs
4. Click **Apply All Fixes** for auto-fix
5. Or use **Review & Fix** to step through each

---

## Features

✅ Three-tab result display  
✅ Auto-fix for common issues  
✅ Step-by-step review mode  
✅ Line-by-line error details  
✅ Validation on save option
